Environment Variables

    

        Configuration is loaded from environment variables. Create a .env file in the project root:
    

    
    
# SMTP Configuration
RADAR_SMTP_HOST=smtp.gmail.com
RADAR_SMTP_PORT=587
RADAR_SMTP_USER=your-email@gmail.com
RADAR_SMTP_PASS=your-app-password
RADAR_EMAIL_FROM=Your Name <your-email@gmail.com>

# IMAP Configuration
RADAR_IMAP_HOST=imap.gmail.com
RADAR_IMAP_PORT=993
RADAR_IMAP_USER=your-email@gmail.com
RADAR_IMAP_PASS=your-app-password

# System Configuration
RADAR_DAILY_LIMIT=100
RADAR_MIN_DELAY=30
RADAR_MAX_DELAY=120
RADAR_SCAN_INTERVAL=12

# Dry Run Mode (set to 1 to test without sending emails)
RADAR_DRY_RUN=0
