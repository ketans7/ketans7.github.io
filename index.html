Business Radar — Automated Lead Generation & AI Outreach Platform

End-to-end system that finds local businesses, analyses their web presence with AI, and runs a complete personalised email outreach pipeline — built solo in Python.

PythonFlaskSQLiteLocal LLMs (Ollama) Web ScrapingSeleniumSMTP / IMAPAutomation
  • Multi-source business discovery: modular scrapers for Google Maps/Places, Yell, Yelp, and FreeIndex behind a common scanner engine, with deduplication keys, lead-quality validation, and rate limiting.
  • AI-personalised outreach at zero API cost: locally hosted LLMs analyse each business's website (with Selenium screenshot capture) and generate tailored cold emails, then automate the full lifecycle — SMTP sending with safety limits, day-3/7/12 follow-ups, and IMAP reply tracking with classification.

RiderBro — AI Decision Assistant for Delivery Riders

Telegram bot that turns a screenshot of a Deliveroo order offer into an instant accept / borderline / skip verdict, grounded in the real cycling route — plus a React operator dashboard over the collected data.

PythonFastAPIReactTelegram Bot OCR + Local VLMMapbox RoutingSQLiteGeospatial
  • Multimodal screenshot understanding: OCR with a local vision-language model fallback (MLX / Ollama) extracts the payout and every pickup/dropoff address fully on-device — screenshots never leave the rider's machine.
  • Real-time decision engine: computes the actual cycling route via Mapbox through every stop, prices in per-stop and peak-hour friction, and returns a verdict with effective £/hour and a per-leg time breakdown in seconds.
  • Self-improving data flywheel: every order, route, outcome, and post-delivery survey is logged to SQLite, powering positioning suggestions from OpenStreetMap venue data and a React analytics dashboard with live feed, spatial heatmaps, and hunting-ground maps.

Code: github.com/ketans7/RiderBro  ·  Live demo: ketans7.github.io/demo/riderbro/
